# MySelectShop

## 요구사항
1. 키워드로 상품 정보 검색
   1. 네이버 쇼핑 API 이용
   2. 상품 이름, 링크 URL, 이미지 URL, 최저가
2. 관심 상품 등록하기
    1. DB에 상품 정보 입력(insert)
       1. 상품 이름, 링크 URL, 이미지 URL, 최저가
       2. 희망 최저가(myprice) -> 0원으로 설정
3. DB 업데이트
   1. 등록된 관심상품의 희망 최저가만 업데이트
4. 관심상품 조회
   1. 등록된 모든 관심상품 조회
      1. 등록된 모든 관심상품 정보 조회
   2. UI에 최저가 표시 조건
      1. 희망하는 최저가보다 실제 최저가가 낮은 경우
